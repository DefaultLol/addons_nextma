Creation de produit de type pack

Methode de creation de Pack

Declaration de pack

Il est utilise pour specifier que le produit cree est un pack dans lequel d'autres produits peuvent etre ajoutes.

Liste de produits

Lors de l'activation de l'option de pack, un nouvel onglet "Produits" s'affiche, dans lequel les produits que vous souhaitez inclure dans le pack sont ajoutes. Remarque: Les packs ne peuvent pas etre ajoutes.

Prix base sur les composants

Si vous voulez que le prix du forfait soit la somme de ses composants, activez l'option "Prix base sur les composants", sinon vous devrez mettre le cout dans "Prix de vente".

Pack dans les commandes de clients

Ajouter des packs dans une commande client

Lors de l'ajout d'un pack dans une commande client, celui ci peut etre affiche de deux manieres differentes en fonction de la declaration de prix du pack:

Pack base sur les prix des composants

Si le pack est declare avec un prix base sur les composants, lors de l'enregistrement de la commande client, le sous-total du package sera affiche avec une valeur egale a zero, tandis que les composants du package apparaitront et la valeur attribuee sera sous-totalisee en fonction de la quantite demande et le prix du composant.

Pack base sur le prix de vente

Lorsque le pack n'est pas declare avec un prix base sur les composants, le pack apparaitra avec le prix de vente attribue par l'utilisateur.Lorsque la commande est enregistree, les composants du produit apparaitront mais, contrairement au precedent, ils auront une valeur en sous-total egal a zero chacun.
